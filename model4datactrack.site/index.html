This web page provides automatic distribution and updates for the Eclipse plug-in for Model4_DataCtrack 1.0.0

Plug-in requirements

This plug-in has primarily been tested with Eclipse 2021-09, but let us know if you have any problems. The plug-in runs under Java 11, or higher. This plug-in requires at least Papyrus 5.2.0.

Plug-in installation

To install the Model4_DataCtrack plug-in:

  1. In Eclipse, click on Help -> Install Software...
  2. Click Add.
  3. Enter the following:
    • Name: Model4DataCtrack
    • URL: https://dsi.uclm.es/cloud/modeling/model4datactrack/releases/1/
    and click OK.
  4. Click the checkbox next to "Model4DataCtrack" it to select it, and click Finish.
    Note: Disable Group items by category option.
    Note: Enable Contact all update sites during install to find required software option.
  5. Click Next.
  6. Select the I accept the terms of the license agreement option to accept the license and click Next.
  7. The plug-in is not digitally signed. Go ahead and install it anyway clicking Ok.
  8. Click Yes to make Eclipse restart itself.
